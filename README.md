# PurchaseRegistration
Created in order to register purchase from shein application 

*******************

I used "npx create-react-app nombre-del-proyecto --template typescript" to create the inicial of project in React with TypeScript

*******************

Project will start with "npm start"

*******************

Backend was configured with Node.js, file server was created with the below: 

mkdir backend
cd backend
npm init -y
npm install express fs

*******************

In the backend I create server.js file in order to get and save purchase orders

*******************

orders.json was created in the backend in order to store purchase orders, it was empty at the beginning

"node server.js"

*******************

axios was installed in frontend purchaseregistration 

"npm install axios"

********************

PurchaseOrderApp.tsx into Components folder has a function to load and save information in the frontend
